# 「ふわっと。」QA Checklist

## 日本語IME確認

- [ ] たまご
- [ ] ばんそうこう
- [ ] ぱん
- [ ] ごみ袋
- [ ] ばなな🍌
- [ ] インターンの持ち物
- [ ] USB-C充電器
- [ ] 名刺・身分証

## Bottom Sheet確認

- [ ] キーボード表示時に保存ボタンが隠れない
- [ ] 小さい画面でも日付チップが押せる
- [ ] 日付選択モーダルが開く
- [ ] 明後日以降の日付を選べる
- [ ] 不正な時刻では保存できない
- [ ] Android: 追加Sheetを開いた状態でBackキーを押すとSheetだけ閉じる
- [ ] Android: 詳細Sheetを開いた状態でBackキーを押すとSheetだけ閉じる

## 通知確認

- [ ] 通知権限をリクエストできる
- [ ] 通知を拒否したあと、設定画面から端末の通知設定を開ける
- [ ] テスト通知が届く
- [ ] リマインダー保存時に10秒後・20秒後通知が届く
- [ ] Android: 通知ドロワーで小アイコンが白い泡として表示され、アクセント色が不自然でない
- [ ] Android: 通知音ONの通知が「リマインダー」チャンネルで届く
- [ ] Android: 通知音OFFの通知が「リマインダー（通知音なし）」チャンネルで届く
- [ ] 通知IDがDBに保存される
- [ ] 期限切れcleanup時に通知がキャンセルされる

## UI確認

- [ ] iPhone SE系
- [ ] Android 5インチ台
- [ ] 標準サイズ
- [ ] Pro Max系
- [ ] Android 標準サイズ
- [ ] バブル文字が小さくなりすぎない
- [ ] バブルが透明感を保っている
- [ ] ホーム左上に「ふわっと。」と「忘れる前に、数秒だけ。」が1行ずつ表示される
- [ ] iOS/Androidのランチャーに新しい単体の光沢泡アイコンが表示される
- [ ] 起動時に文字なしの新しい泡スプラッシュが中央表示される
- [ ] Android: ホーム画面やアプリ一覧でランチャーアイコンがマスクされても欠けない
- [ ] Android: 画面下のナビゲーションバーが淡い背景色で、ボタン/ジェスチャー表示が読める
- [ ] 設定画面が崩れない

## リリース順確認

- [ ] Android先行リリース用のAABで起動・通知・SQLiteを確認する
- [ ] Android実機でランチャーアイコンの丸/角丸/テーマ別表示を確認する
- [ ] Android実機でナビゲーションバーの背景色とアイコン視認性を確認する
- [ ] Android実機で追加Sheetと詳細SheetのBackキー閉じを確認する
- [ ] Android実機で通知小アイコンと通知アクセント色を確認する
- [ ] Androidリリースごとに `android.versionCode` を上げる
- [ ] App Store後追いリリースごとに `ios.buildNumber` を上げる
- [ ] Google Playのクローズドテスト要件を満たすテスターを確保する
- [ ] App Store後追い用にiPhone実機でも同じQAを実施する
- [ ] 初回App StoreリリースはiPhone対象として扱い、iPad最適化は別タスクにする
- [ ] Widget実装は別タスクとして扱い、このQAでは対象外にする
